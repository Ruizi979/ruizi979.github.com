Inheritance in CSS

Inheritance in CSS controls what happens when no value is assigned to an attribute on an element.Example can be set a text color of an element, then all the descendants of that element will inherit the same text color.

Specificity in CSS

The browser will follow some rules to determine which one is the most specific rule to win If there are two or more conflicting CSS rules that point to the same element. The particularity is the level that determines which style declarations ultimately apply to an element. Selectors have four specific levels: inline style, ID, class, element, and pseudo-element. A selector is a mode for selecting elements to style in CSS.

ID and Class Attributes

The difference between ID and class attributes is that a class is that ID can be used to identify one element,a class selector is a name preceded by a full stop (“.”) and an ID selector is a name preceded by a hash character (“#”).
