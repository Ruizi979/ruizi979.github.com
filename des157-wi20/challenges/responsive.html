About Media Queries

Media queries are a feature of CSS that enable webpage content to adapt to different screen sizes and resolutions. They are a fundamental part of responsive web design and are used to customize the appearance of websites for multiple devices.The max-width property defines the maximum width of an element.If the content is larger than the maximum width, it will automatically change the height of the element.The min-width property defines the minimum width of an element.If the content is smaller than the minimum width, the minimum width will be applied. we should give different size for ipad, iphone and loptop. So the web page can fit into a comfortable size for each device.

Relative Units

The relative unit size will vary. In the specification, relative length units are defined as em, ex, ch, and rem. These are relative font lengths. The specification also defines% values that are always relative to another value.

Meta Viewport Tag

Viewport meta tags let you simply use the maximum and minimum widths from desktop to mobile. The width property controls the size of the viewport. The CSS pixel width of the screen, the ratio is 100%. The initial-scale property controls the zoom level when the page is first loaded. Without put the meta viewport tag, the media tag can not really work, so it will not show good size for each device.
