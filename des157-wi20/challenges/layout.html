Structure, Phrasing and Display

Block elements appear on the screen as if they have line breaks before and after them. The example can be p tag or h1 tag. An inline element is an element that occupies only the space defined by the tag that defines the element, without disrupting the content flow.The difference between these two elements is the commonly misunderstood web design concepts, but not necessarily so! Basically, inline elements do not cause line breaks (starting from a new line) and do not take up the entire width of the page, they only take up the space defined by their start and end tags.

Box Model

CSS box model include content edge, padding, border, and margin.The box-sizing property allows us to include padding and borders in the total width and height of the element. If you set the size of the box: border-box; including the border of the element and the width and height of the border: the two divs are now the same size

Background Images

The images on the page inside <img> tags has two required attributes: src and alt. Images are linked to HTML pages, not inserted. Images loaded through CSS can only used as background image.
